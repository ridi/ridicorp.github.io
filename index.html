OUR PHILOSOPHY

풍부한 상상, 깊은 통찰로 인류의 정신을 풍요롭게

Rich Imagination, Deep Insight

먹는 것이 우리의 몸을 만들듯, 보고 듣는 것이 우리의 정신을 이룹니다. 풍부한 상상과 깊은 통찰이 인류 정신을 비옥하게 할 자양분이라 믿습니다.

리디는 전자책, 웹툰·웹소설, 애니메이션, 아티클·뉴스로 풍부한 상상과 깊은 통찰을 세상에 건넵니다.

RIDI CODE

‘RIDI CODE’는 우리가 만드는 모든 제품에 녹아 들어 있습니다. 전자책, 웹툰, 웹소설, 만화, 애니메이션 스트리밍, IT뉴스까지 ‘인류의 정신을 풍요롭게’ 만들기 위한 리디의 끝 없는 확장과 도전의 바탕에는 RIDI의 핵심가치와 일하는 방식이 있습니다.

  • RIDI CODE 1

    본질에 집중, Essence

    본질을 향한 질문을 멈추지 않을 때, 물음표는 느낌표가 됩니다. 우리는 언제나 본질에 집중합니다.

  • RIDI CODE 2

    고객 중심, Customers

    고객의 목소리는 호통이며, 칭찬이고, 훌륭한 조언입니다. 우리는 고객을 중심으로 길을 찾습니다.

  • RIDI CODE 3

    건강함 추구, Sound body & mind

    우리는 건강한 몸과 마음으로 세상에 이로운 RIDI 생태계를 만듭니다.

  • RIDI CODE 4

    세계적 관점, Global Perspective

    풍부한 상상과 깊이 있는 통찰에는 국경이 없습니다. 우리는 ‘세계인을 위한 RIDI’를 지향합니다.

  • RIDI CODE 5

    끈기, Grit

    열 번 찍어 안 넘어가는 나무도 있습니다. 우리는 열 한 번을 찍는 투지로 끊임없이 도전합니다.
